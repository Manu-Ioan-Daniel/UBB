Un algoritm aleator pentru care rezultatele obtinute sunt corecte doar cu o anumita probabilitate se numeste algoritm Monte Carlo.
- Se examineaza probabilitatea cu care rezultatul este corect; probabilitatea de eroare poate fi scazuta semnificativ prin executii repetate,independente

---

**Exemplu:** Testul Miller-Rabin,care verifica daca un numar natural este prim sau este numar compus;testul returneaza fie raspunsul "numarul este sigur un numar compus" sau raspunsul "numarul este probabil un numar prim"
