â–¶ Extragerea cu returnare este folosita in **metoda bootstrap** (engl. bootstrapping), care este o metoda utilizata pentru a estima proprietatile statistice dintr-un set de date.
* Tehnica implica **re-esantionarea** (engl. resampling), folosind datele dintr-un singur set de date cu $n$ observatii.
* Un **set de date bootstrap** este format din $n$ observatii alese aleator **cu returnare** si independent din setul de date initial.
Bootstrapping este o procedura statistica care re-esantioneaza un singur set de date pentru a crea mai multe esantioane (folosind simulari). Aceste esantioane sunt folosite pentru a face inferente statistice asupra setului initial de date.
