
### Definitie

**Definitie:** Functia de repartitie $F : \mathbb{R} \to [0, 1]$ a unei variabile aleatoare discrete $X$, care ia valorile $\{x_i, i \in I\}$, este
$$
F(x) = P(X \le x) = \sum_{i \in I: x_i \le x} P(X = x_i) \quad \forall x \in \mathbb{R}.
$$
În l. engleza denumirea este *cumulative distribution function*, prescurtat cu **cdf**.

### Exemplu 1

Functia de repartitie $F_X : \mathbb{R} \to [0, 1]$ a v.a. discrete $X$ este
$$
F_X(x) = P(X \le x) =
\begin{cases}
0, & \text{daca } x < -2 \\
0.5, & \text{daca } -2 \le x < 1 \\
0.7, & \text{daca } 1 \le x < 2 \\
1, & \text{daca } 2 \le x
\end{cases}
$$
Determinati valoarea medie a lui $X$. 

$$
P(X = -2) = 0.5,\quad

P(X = 1) = 0.7 - 0.5 = 0.2 ,\quad

P(X = 2) = 1 - 0.7 = 0.3
$$
$$
E(X) = -2 \cdot 0.5 + 1 \cdot 0.2 + 2 \cdot 0.3 = -0.2
$$

## Proprietati ale Functiei de Repartitie

Functia de repartitie $F$ a unei variabile aleatoare discrete $X$ are urmatoarele proprietati:

(1) $F(b) - F(a) = P(X \le b) - P(X \le a) = P(a < X \le b) \quad \forall a, b \in \mathbb{R}, a < b$.
(2) $F$ este **monoton crescatoare**, adica pentru orice $x_1 < x_2$ rezulta $F(x_1) \le F(x_2)$.
(3) $F$ este **continua la dreapta**, adica $\displaystyle \lim_{x\searrow x_0} F(x) = F(x_0) \quad \forall x_0 \in \mathbb{R}$.
(4) $\lim_{x\to\infty} F(x) = 1$ si $\lim_{x\to-\infty} F(x) = 0$.

**Observatie:**
* Orice functie $F : \mathbb{R} \to \mathbb{R}$, care are proprietatile (2), (3), (4)  este o functie de repartitie.
* Functia de repartitie a unei v.a. descrie complet comportamentul probabilistic al acelei v.a.

## Functia de Repartitie in Python (scipy.stats.cdf)

În Python: se calculeaza $F(x) = P(X \le x)$

* pentru $X \sim Bino(n, p)$ cu `scipy.stats.binom.cdf(x, n, p)`,
* iar pentru $X \sim Hyge(n, n_1, n_2)$ cu `scipy.stats.hypergeom.cdf(x, n1+n2, n1, n)`.

```python
# Fie o urna cu 10 bile, din care 5 sunt rosii  
# X (v.a.) = cate bile rosii au fost extrase in 5 extrageri cu returnare  
# se reprezinta grafic functia de repartitie a lui X  
import scipy.stats  
import matplotlib.pyplot as plt  
import numpy as np  
n=5  
p=0.5  
x = np.linspace(-3, n+3, 1000)  
y=scipy.stats.binom.cdf(x, n, p)  
plt.plot(x, y, "r.") #r vine de la rosu, "."inseamna ca vreau un punct individual,nu vreau sa fie unite intre ele cu o linie  
for t in range(n+1):  
    plt.plot(t, scipy.stats.binom.cdf(t, n, p), "ko")  
    plt.plot(t, scipy.stats.binom.cdf(t-1/1000, n, p), 'ko', mfc='none')  
plt.xlabel("x")  
plt.ylabel("F(x)=P(X <= x)")  
plt.title("Functia de repartitie a lui X")  
plt.xticks(range(-3, n+4))  
plt.grid()  
plt.show()
```
